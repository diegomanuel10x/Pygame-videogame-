# Pygame-videogame-
This is my first game, I used pygame to make it. The game is about a pig that is trying to escape from a bunch of predators.
I used Canva to design all the game elements. I provided all the characters' png for anyone who woud like to code along.

I followed the "The ultimate introduction to Pygame" tutorial by Clear Code, with a few adjustmenst of my own to give the its own personal and unique attributes.
